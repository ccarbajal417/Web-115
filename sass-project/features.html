ST Package 1

The ST package 1 is 6 speed and has a cloth interior, it comes with the basic SYNC sound system and still has a sleek and attractive look to it. Also comes with push to start, and backup camera.

ST Package 2

The ST package 2 has more luxurious look to it with all leather, heated, Recaro seats. Also a Sony sound system all round, with a push to start button, backup camera, and navigation system.

ST Package 3

The ST package 3 has more luxurious look to it with all leather, heated, Recaro racing seats. Also a Sony sound system all round, with a push to start button, backup camera, and navigation system, retractable side mirrors, custom carbon fiber manual shifter.
